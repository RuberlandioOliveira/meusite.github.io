Bem-vindo ao Site de Teste Ruberlandio

Estamos trabalhando para em breve oferecer projetos de sites personalizados para você.
Este site está sendo construido ultilizando HTML,CSS e JS que são linguagem de marcação.
HTML (Linguagem de Marcação de Hipertexto) é responsável pela estrutura do conteúdo,
definindo elementos como parágrafos, títulos, listas, imagens e formulários. É como o esqueleto de uma página web.
CSS (Folhas de Estilo em Cascata) define a aparência visual da página, como cores, fontes, tamanhos,
espaçamento, entre outros elementos. É como a pele e a roupa de uma página web.

JavaScript é uma linguagem de programação que permite adicionar funcionalidades interativas
e dinâmicas à página, como animações, tratamento de eventos,validação de formulários e interações com o usuário.
É como os músculos e o sistema nervoso de uma página web, dando-lhe movimento e reação.
